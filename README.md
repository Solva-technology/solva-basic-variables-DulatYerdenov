# Упражнение на переменные и базовые операции в Python

Это упражнение поможет вам освоить создание и использование переменных, а также выполнение базовых математических операций.

## Цель

Ваша задача — отредактировать файл `main.py`, чтобы он решал несколько простых задач, и затем запустить тесты для проверки.

## Инструкция

### Шаг 1: Настройте окружение

Как и в предыдущем задании, установите виртуальное окружение и зависимости:

```bash
python -m venv venv
source venv/Scripts/activate
pip install -r requirements.txt
```

### Шаг 2: Выполните задания в `main.py`

Откройте файл `main.py` и следуйте инструкциям в комментариях. Вам предстоит:

1.  **Создать переменные** для вашего имени и возраста и вывести их.
2.  **Выполнить простые расчеты**: посчитать, сколько вам будет лет через 10 лет.
3.  **Решить задачу**: рассчитать, сколько яблок останется у вас после того, как вы поделитесь с другом.

### Шаг 3: Запустите тесты

После выполнения заданий запустите тесты, чтобы проверить правильность решений:

```bash
pytest
```

Если все сделано верно, вы увидите сообщение об успешном прохождении тестов. В противном случае `pytest` укажет на ошибки.
